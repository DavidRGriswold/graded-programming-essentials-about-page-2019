Computer Programming Principles

What will you do?

  • Learn the basic principles of computer programming using Python, one of the most versatile and powerful computer languages around. Press the button below to see a very simple Python program in action.

  • Create applications to solve realistic problems
  • Use professional programming tools, such as git, Integrated Development Environments, and professional coding editors
  • Learn how computers "think,"" store their data, and interact with each other
  • Learn about the major parts of a computer and how they interact

What do you need to know?

  • This course has no prerequisites. Come one, come all!
  • This course will focus primarily on coding to do things and solve problems. If you are more interested in design and presentation, you should consider taking the Web Design course (or take both!)
